# ASSET_PAGES.md — 資産ページ管理台帳

このファイルは、ニュース記事とは異なり**長期運用・毎年育てる「資産ページ」**（常設ガイド・まとめページ）を一覧管理するための台帳です。
ニュース記事の一覧・公開状況は`data/articles/`、運営ルール全般は[PROJECT_RULES.md](../PROJECT_RULES.md)を参照。

## 運用ルール

- 新しい資産ページを作成したら、該当カテゴリの表に追記する
- ページ内容を更新したら「公開日」欄はそのまま、必要に応じて備考に更新履歴を書く（詳しい経緯は[DECISIONS.md](../DECISIONS.md)へ）
- 「資産候補（未作成）」に挙げた案を実装したら、該当カテゴリの表に移動し、候補一覧から削除する
- 優先度は目安。Search Consoleの実績等を踏まえて随時見直す
