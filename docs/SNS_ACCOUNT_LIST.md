# SNS_ACCOUNT_LIST.md — フォロー・交流アカウント管理台帳

このファイルは公式X（[@TakaTodayJP](https://x.com/TakaTodayJP)）のフォロー先・フォローバック状況を管理する台帳。
運用方針（フォロー方針・返信方針等）は [SNS_OPERATION.md](./SNS_OPERATION.md) を参照。

## 運用ルール

- 今後フォローしたアカウントは、該当カテゴリの表に追加する
- 「フォロー済」「フォローバック」は `済` / `未` / `－`（対象外・確認中）で記入する
- サンプル行は実在確認済みのアカウントのみ記載（2026-07-04時点、宝塚市公式サイト「ソーシャルメディア一覧」等で確認）。未確認のまま推測でIDを記入しない
- 空欄の行・テーブルは今後の追加用。件数を埋めることを目的化せず、実際に交流のある/価値のあるアカウントのみ追加する
- 新しく追加する行の「優先度」は、下記「優先度の目安」を参照して判断する
